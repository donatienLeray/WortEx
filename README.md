# WortEx
Linguistic gaming with Python Final Project. A german word game heaviliy inspired by https://wordhub.com

### Dependencies

- Python version >= 3.6
  check with `python --version`

- if you have a unix system, you will need  or rgrep
  check with `grep --version`
